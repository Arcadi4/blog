其实基本上这篇文章存在的意义只是因为我不想在首页塞三篇空文章，那样很难看。

## 技术栈

这个网页使用的技术栈：

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
- Vibe Code
- Figma

是的，看得出来这个列表中的*某一项*才是最重要的。

## 关于网页设计

这次的网页设计基本遵循以下几个目标：

- 使用尽可能大胆、难以掌握的配色
- 避免在UI中使用任何图标（icon）
- 展现最纯粹的字形和排版美学

## 内容方向

我自认为不是一个在技术上极其出众的开发者（字面意思，我不是在炫耀）。所以实际上不会如预期的非常技术，除技术之外可能还会包括游戏、平面设计、甚至社会科学等等。
